Проект сделан на основе паттерна PageObject

В папке Yandex Entities класс YandexHomePage - PageObject страница, в ней указаны взаимодействия с элементами и методы.
						класс YandexHomePageSettings - хранит настройки для нашего класса (локаторы, урл, вводимые данные в форму)

В папке Test Entities   класс TestSettings - хранит настройки для тестов (инициализация необходимых элементов, implicit wait, закрытие драйвера после каждого прогона)
						класс YandexTests -  хранит логику тестов.

Класс AllTypeOfLocators хранит нахождение локаторв с разными методами класса By.
